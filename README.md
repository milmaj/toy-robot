# toy-robot

### Tech stack

- Typescript
- React.js
- Tailwind
- npm/yarn

### Install dependencies

`yarn`

### Run

`yarn start`

### Basic keyboard shortcuts:

- P: Place (x: number, y: number, direction?: Direction)
- M: Move
- <: Left
- >: Right
- R: Report
